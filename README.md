# 개체 만들기

## 사용 방법

1. Visual Studio에서 `Chapter25-CreatingObjects.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice | 교안 코드 따라치기 (13개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| GameCharacter | 게임 캐릭터 육성 |
| BankAccount | 은행 계좌 관리 |
| ProductInventory | 상품 재고 관리 시스템 |

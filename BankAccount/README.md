# 은행 계좌 관리

은행 계좌를 관리하는 `BankAccount` 클래스를 작성하시오.

**요구사항**

- 정적 멤버
  - `TotalAccounts`: 전체 계좌 수 (static int)
  - `ShowTotalAccounts()`: 전체 계좌 수를 출력하는 정적 메서드
- 인스턴스 멤버
  - `AccountNumber`: 계좌 번호 (string)
  - `OwnerName`: 예금주 이름 (string)
  - `Balance`: 잔액 (int)
  - `Deposit(int amount)`: 입금 메서드
  - `Withdraw(int amount)`: 출금 메서드 (잔액 부족 시 경고)
  - `ShowInfo()`: 계좌 정보 출력

**동작 설명**

- 계좌가 생성될 때마다 `TotalAccounts`가 1씩 증가해야 함
  - 힌트: 생성자를 사용하거나, 메인에서 수동으로 증가시킴
- 출금 시 잔액이 부족하면 "잔액이 부족합니다" 메시지 출력

## 예상 실행 결과

```
=== 계좌 개설 ===
[1001] 홍길동 계좌가 개설되었습니다.
[1002] 이순신 계좌가 개설되었습니다.
현재 총 계좌 수: 2개

=== 거래 ===
[1001] 50000원 입금 완료. 잔액: 150000원
[1002] 30000원 입금 완료. 잔액: 80000원
[1001] 200000원 출금 실패. 잔액이 부족합니다.
[1001] 100000원 출금 완료. 잔액: 50000원

=== 최종 계좌 정보 ===
계좌번호: 1001, 예금주: 홍길동, 잔액: 50000원
계좌번호: 1002, 예금주: 이순신, 잔액: 80000원
```
